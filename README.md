## ‚ú® Features / Highlights

### **Business Problem**
Consumers and retailers often struggle to compare smartphone specifications across brands and models efficiently.

### **Goal of the Dashboard**
To provide an interactive tool that:
- Displays detailed specs for selected models.
- Allows quick comparison across brands.
- Highlights key features like memory, storage, battery, and price.

### **Walkthrough of Key Visuals**
- **Top Navigation Tabs**:
  - Brands: Apple, ASUS, Google, Honor, Huawei, Motorola, OnePlus, Oppo, Realme, Samsung, Xiaomi.

- **Slicer (Model Selection)**:
  - Dropdown to select specific models (e.g., Apple iPhone SE 2022).

- **Key Specs Cards**:
  - **Memory**: Up to 4 GB
  - **Storage**: Up to 256 GB
  - **Launch Year**: 2022
  - **Display**: 4.7" Retina IPS LCD
  - **Resolution**: 750 x 1334
  - **Battery**: 2018 mAh
  - **Camera**: 4K 24/30/60fps
  - **Pixel Density**: 326 PPI

- **Variant & Price Table**:
  - Lists storage variants and prices from different stores (Amazon DE, UK, US).

- **Feature Highlights**:
  - Fingerprint sensor, NFC, Compass, Barometer, Accelerometer.

- **Processor & GPU Info**:
  - Apple A15 Bionic (5 nm), Hexa-core CPU, Apple GPU (4-core graphics).

- **Dynamic Font Color**:
  - Changes based on selected phone color.
